The new flexbox layout mode is poised to redefine how we do layouts in CSS. While there are still some browsers that do not support flexbox without prefixes, support is getting better and better and there are some amazing things we can do with regards to layout, using flexbox. This page layout is an example of a layout using flexbox.
I am a flex item, inside of a flex container. I am sitting inbetween two other flex items. We are all evenly spaced within our container, as we are set to be equal, using CSS flexbox syntax.
