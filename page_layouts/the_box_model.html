The Box Model

The problem with the box model is that when you add padding and border to an element, it increases the width and height in addition to the specified width and height. This can be a pain when you have complicated layouts with lots of elements to take into consideration.

This div is 200px wide and 100px high. There is no padding or border applied, which doesn't look great, but the element maintains its specified dimensions.
This div also has a 200px width and 100px height applied, but there is a 1px border and 20px of padding, making the div 242px wide and 142px high.
