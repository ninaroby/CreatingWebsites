Position

The position property is important to understand, if you are going to create effective page layouts. There are 4 properties that you should understand well.

  • Static
  • Relative
  • Fixed
  • Absolute

Static

Static is the default value for positioning. If you don't position an element it is "static" and therefore "not positioned"

Relative

When you set an element with the position of relative, it will be relative to it's own natural place in the document:"relative to itself". So this div is set to position:relative
top:10px
left:10px
This means that it will display 10px down and 10px left from where it would be in the document flow.

Fixed

This is a tricky one for use on the mobile platform. Essentially, if you position an element as fixed, it will stick to the screen window in that position. The element stuck to the bottom, right of the screen is 'position: fixed' and no matter how much you scroll, it will remain there!
Hi, there, I am fixed here! Still here!

Absolute

The box on the right is set to an absolute position of:
top:-10px
right:15px

This is where you can control the exact position of an element, relative to the nearest positioned element.
