Learn CSS Layout

The "display" property is an important property in CSS. All elements have a default display property, which is usually "block" or "inline", depending on the element.

Block

Block level elements start on a new line and stretch out as far as possible, depending on available space. Here is a list of elements, which are default block level elements:

  • div
  • p
  • form
  • section
  • article
  • header
  • footer
  • aside
  • h1 through h6 are all block level elements

Inline

An inline level element can wrap text inside a paragraph, without disturbing the document flow. For example this sentence is surrounded with an "em" tag. Here are some inline level elements:

  • b, big, i, small, tt
  • abbr, acronym, cite, code, dfn, em, kbd, strong, samp, var
  • a, bdo, br, img, map, object, q, script, span, sub, sup
  • button, input, label, select, textarea

None

The script tag, has a default property of display:none. This display property is commonly used with javascript, to allow an element to hide and show, depending on the situation. It should not be confused with visibility:hidden: this is where the element still takes up space, but is hidden from view. The value "none" renders an element as though it does not exist.
