The Clearfix Hack

Uh oh... this image is taller than the element containing it, and it's floated, so it's overflowing outside of its container!

This is much better! You simply use the hack: overflow: hidden;

It works for modern browsers. If you want to support IE6 you will want to add the following: Overflow:auto; zoom:1;

This is a class exercise for the CIT-85 class. The images and information are sourced from Learn CSS Layout at learnlayout.com
